# Finding card template

Copy one block per finding. Max 15 findings per pass file.

```markdown
### [P0|P1|P2] Título curto
- **Pass:** XX
- **Superfície:** Resource → Operation
- **Arquivos:** path/file.ts:line, path/file.ts:line
- **Sintoma:** o que o usuário vê ou o que quebra
- **Evidência:** trecho/comportamento observado no código
- **Fix proposto:** mudança concreta
- **Esforço:** S | M | L
- **Decisão:** FIX | DEFER | ACCEPT
```
